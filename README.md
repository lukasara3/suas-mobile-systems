# 📱 Mobile Systems - HS Schmalkalden

This repository contains practical exercises and Android applications developed during the **Mobile Systems** course at Schmalkalden University of Applied Sciences (SUAS) in Germany.

The main objective of these projects is to understand the core concepts of mobile application development, UI/UX principles, and device hardware integration using the Android ecosystem.

### 🛠️ Tech Stack & Tools
*   **Environment:** Android Studio
*   **Languages:** Java / Kotlin / XML
*   **Build System:** Gradle
